# React Router Localization

Monorepo for React Router localization packages.

## Packages

| Package | Description | Docs |
|---------|-------------|------|
| [i18next-react-router](./packages/i18next) | Seamless i18next integration for React Router v7+ | [README](./packages/i18next/README.md) |

## Quick Start

```bash
npm install i18next-react-router i18next react-i18next
```

For detailed documentation, see the [i18next-react-router README](./packages/i18next/README.md).
